# UNA AlgProg - Lista 04

Repositorio com a quarta lista de exercicios de Algoritmos e Programacao, implementada em Portugol (`.por`).

O projeto reune 5 programas simples com foco em estruturas condicionais, validacoes e classificacao de dados de entrada.

## Conteudo

| Arquivo | Tema | Descricao |
| --- | --- | --- |
| `01_validador_idade.por` | Maioridade | Verifica se a idade informada permite acesso. |
| `02_radar_velocidade.por` | Multa por velocidade | Calcula excesso acima de `80 km/h` e o valor da multa. |
| `03_sistema_notas.por` | Media escolar | Classifica o aluno em aprovado, recuperacao ou reprovado. |
| `04_par_impar.por` | Paridade | Informa se o numero digitado e par ou impar. |
| `05_classificador_triangulos.por` | Triangulos | Valida lados e classifica o triangulo informado. |

## Objetivo da lista

Praticar conceitos basicos de logica de programacao:

- leitura de dados com `leia`
- exibicao de mensagens com `escreva`
- decisoes com `se`, `senao` e `senao se`
- operadores relacionais, logicos e aritmeticos

## Como executar

1. Abra o arquivo desejado em uma ferramenta compativel com Portugol, como o `Portugol Studio`.
2. Execute o programa pela funcao `inicio()`.
3. Informe os valores solicitados no console de entrada.

## Estrutura do repositorio

Todos os exercicios estao na raiz do projeto e cada arquivo representa uma solucao independente.

```text
.
|-- 01_validador_idade.por
|-- 02_radar_velocidade.por
|-- 03_sistema_notas.por
|-- 04_par_impar.por
|-- 05_classificador_triangulos.por
`-- README.md
```

Este repositorio tem perfil academico e foi organizado como uma lista de pratica, nao como uma aplicacao unica.
